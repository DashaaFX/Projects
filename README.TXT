# 🏀 Basketball Target Practice Game

A JavaFX-based aim training game where players test their reflexes by clicking on moving basketballs. Built as a final project for **CSC 233 (Fall 2024)**.  

## 🎮 Game Features
- **3 Difficulty Levels**: Easy, Medium, Hard  
  - Easy → 10 good balls, 10 bad balls  
  - Medium → 20 good balls, 20 bad balls  
  - Hard → 30 good balls, 30 bad balls  
- **Win Condition**: Click all the red basketballs without clicking any black balls.  
- **Dynamic Ball Controls**: Adjust ball size with a slider.  
- **Randomized Motion**: Basketballs move randomly across the court.  
- **Performance Tracking**: Displays hit/miss percentage and timer.  
- **Menu Options**:
  - About the developer  
  - Instructions  
  - Difficulty information  
  - Quit  
 

## ⚙️ How to Play
1. Start the game by clicking **"Start New Game"**.  
2. Select a **difficulty level** from the dropdown.  
3. Use your mouse to click on **red basketballs** (good balls).  
4. Avoid clicking **black basketballs** (bad balls).  
5. Track your **hit/miss ratio** and **time** at the bottom of the screen.  
6. Adjust the **ball size slider** to make the game easier or harder.
